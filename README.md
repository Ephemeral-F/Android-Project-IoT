DEMO
===========================

### 环境依赖

Android 4.0及以上

### 部署步骤

1. 下载app-debug.apk至安卓设备

2. 运行并安装，进入主界面
   
3. 对于蓝牙调试，点击搜索设备，选择一个已配对的设备，进行通讯、调试

4. 对于WiFi调试，输入服务器IP地址和端口地址，进行通讯、调试
